Ugledna poslovna žena stavlja svoju karijeru i obiteljski život na kocku kada započne burnu aferu sa svojim mnogo mlađim pripravnikom. Priča donosi uvjerljivu naraciju s iznimnom kinematografijom i snažnim izvedbama.
